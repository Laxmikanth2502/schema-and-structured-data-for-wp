== Description ==
Schema and Structured Data for WP is a Google Rich Snippet plugin to get best results in structuring your websites for the Search Engines.

= Features: =

* Conditional Display Feilds: Meaning you include or exclude any posts, pages, post types, taxonomies and more! 
* Knowlegde Base Support
* Unlimited Custom Post Types
* Easy and Extend the Minimal Setting
* Automatically Work in AMP or NON-AMP
* Archive Page Listing Support
* JSON-LD Format
* Breadcrumb Listing Support
* Advance Custom Schema Types
